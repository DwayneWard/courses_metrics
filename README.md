# Аналитика образовательной эффективности курса

Универсальный инструмент для анализа ведомостей оценок курса с расчетом метрик образовательной эффективности.

## Структура проекта

- `course_analytics.py` - основной модуль с классом `CourseAnalytics` для анализа данных
- `course_analytics.ipynb` - Jupyter notebook для интерактивного анализа и визуализации
- `CONFIG_GUIDE.md` - гайд по конфигурированию под разные ведомости курсов

## Установка

1. Создайте виртуальное окружение (если еще не создано):
```bash
python3 -m venv venv
source venv/bin/activate  # для macOS/Linux
# или
venv\Scripts\activate  # для Windows
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Использование

#### 1. Откройте `course_analytics.ipynb`

#### 2. Настройте конфигурацию курса

В ячейке с конфигом (`COURSE_CONFIG`) задайте как минимум:

- путь к ведомости `excel_path`;
- по желанию — `activity_rules` (если нужно объединить столбцы) и `activity_types` (если требуется добавить свои паттерны).

Подробная инструкция: `CONFIG_GUIDE.md`.

#### 3. Запустите все ячейки

Используйте `Cell > Run All` или запускайте ячейки последовательно.

#### 4. Проверка маппинга

В ноутбуке есть ячейка маппинга, она позволяет проверить правильно ли загрузились и "разобрались" по типам работы

Если таблица пустая или ожидаемые активности отсутствуют, поправьте конфигурацию.

#### 5. Результаты

Все метрики будут рассчитаны и визуализированы автоматически

## Формат входных данных

Excel файл должен иметь общую структуру ведомостей ЦУ, которые выгружаются из LMS

## Рассчитываемые метрики

### Метрика 1: Завершенность и активность
- Процент завершения по каждому типу работы
- Траектория вовлеченности студентов
- Выявление точки отсева

### Метрика 2: Успеваемость
- Распределение оценок по типам работ
- Прогресс студентов (динамика оценок)
- Сводная статистика по прогрессу

### Метрика 3: Качество обучения
- Корреляция между типами работ
- Консистентность результатов студентов

### Метрика 4: Сравнительные метрики
- Сравнение сложности проектов
- Анализ вариативности оценок

## Экспорт результатов

В notebook есть возможность экспортировать все метрики в Excel файл:
```python
export_to_excel = True
```

Все метрики будут сохранены в отдельные листы Excel файла.
